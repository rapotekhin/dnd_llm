# Release Notes

## Version 0.2.0

### Новые функции

- **Система поднятия уровня персонажа**
  - Добавлен экран поднятия уровня с пошаговым выбором новых возможностей
  - Проверка опыта для определения возможности поднятия уровня
  - Подсветка кнопки "Персонаж" золотым цветом, когда доступно поднятие уровня
  - Выбор новых особенностей (features) с поддержкой выбора подособенностей (subfeatures)
  - Улучшение характеристик (ability score improvements)
  - Выбор новых заклинаний и заговоров с ограничениями согласно правилам D&D 5e
  - Выбор новых навыков (proficiency choices) только из данных уровня
  - Экран подтверждения с итоговой информацией о выборе
  - Автоматическое обновление `class_specific` характеристик (action_surges, sorcery_points, etc.)

### Улучшения UI

- **Исправление перекрытия элементов интерфейса**
  - Стандартизирован порядок отрисовки элементов (z-order) во всех экранах
  - Модальные окна теперь рисуются после всех элементов с затемняющим overlay
  - Тултипы всегда рисуются последними, поверх всех элементов
  
- **Адаптивное позиционирование элементов**
  - Счетчики способностей: кнопки "+/-" теперь позиционируются динамически после текста метки
  - Экран "Биография": метки размещаются выше соответствующих кнопок и полей ввода
  - Список мировоззрений: ограничен высотой и не выходит за пределы экрана
  - Списки выбранных элементов (заклинания, навыки): позиционируются ниже списков доступных элементов
  - Все элементы адаптивно позиционируются с учетом границ экрана и кнопок навигации

- **Исправления в экранах**
  - Экран "Способности": текст больше не перекрывает кнопки "+/-"
  - Экран "Биография": текст меток не перекрывает кнопки и поля ввода
  - Экран поднятия уровня: текст заголовков не перекрывается с индикаторами этапов
  - Модальные окна выбора подособенностей: добавлена прокрутка для длинных списков
  - Все экраны: элементы ограничены областями, чтобы не перекрывать кнопки навигации

### Технические улучшения

- Добавлен класс `LevelUpBuild` для управления выбором при поднятии уровня
- Добавлены утилиты `level_up_utils.py` для проверки возможности поднятия уровня
- Улучшена обработка `class_specific` данных в `Character` и `CharacterBuild`
- Добавлено клиппирование текста в панелях информации
- Улучшена обработка событий в модальных окнах с поддержкой прокрутки
